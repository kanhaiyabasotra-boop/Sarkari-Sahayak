Aadhar Card Mein Address Online Kaise Badlein

UPDATED: JULY 2026 · 5 MIN READ

Agar aap shift ho gaye hain ya naya ghar liya hai, toh Aadhar card ka address update karna zaroori hai — bank KYC, naya SIM, ya kisi bhi sarkari kaam ke liye current address match karna padta hai. Achhi baat yeh hai ki yeh poora process ghar baithe, online ho jaata hai.

Yeh Documents Taiyar Rakhein

  • Registered mobile number (jo Aadhar se linked ho)
  • Naye address ka proof (bijli bill, bank statement, ya rent agreement)
  • Scanned copy (PDF, size 2MB se kam)

Step-by-Step Process

  1. UIDAI ki website par jayein

    Apne browser mein UIDAI ki official self-service update website open karein aur "Update Aadhar" wale option par click karein.

  2. Aadhar number aur OTP se login karein

    Apna 12-digit Aadhar number daalein. Registered mobile par ek OTP aayega, usse verify karein.

  3. "Address" option select karein

    Update options mein se sirf "Address" ko tick karein — baaki fields ko khali chhod dein taaki sirf address hi update ho.

  4. Naya address bharein aur proof upload karein

    Naya address carefully type karein (jaisa proof document mein likha hai, waise hi), aur scanned document upload karein.

  5. Submit karein aur URN save karein

    Request submit karne ke baad ek Update Request Number (URN) milega — isse status check karne ke liye sambhal kar rakhein.

✓

Verified July 2026: Yeh process UIDAI ke current self-service portal ke hisaab se hai. Update ho jaane mein aam taur par 7–15 din lagte hain.

Status Kaise Check Karein

Apna URN number UIDAI portal ke "Check Aadhar Update Status" section mein daal kar, aap kabhi bhi apni request ki current state dekh sakte hain — pending, processing, ya approved.
